## ✅ STEP 1　インストール・起動

### 手順
- [ ] ブラウザで **「CapCut PC」** と検索
- [ ] 公式サイト（capcut.com）からダウンロード
- [ ] インストーラーを起動してインストール
- [ ] アカウント登録 または Googleアカウントでログイン

<a id="step2"></a>
## ✅ STEP 2　新規プロジェクトの作成

### 手順
- [ ] CapCutを起動
- [ ] 左上の **「新しいプロジェクト」** をクリック
- [ ] 画面比率を選ぶ

### 📐 どの比率を選ぶ？

| 比率 | 使う場面 | isla での用途例 |
|------|---------|----------------|
| **9:16（縦）** | リール・TikTok・ストーリーズ | イベント告知・ボランティア募集 |
| **16:9（横）** | YouTube・スクリーン投影 | インタビュー動画・説明動画 |
| **1:1（正方形）** | Instagramフィード | 活動報告・写真まとめ |

### 🎯 実際にやってみよう
> 今回は **9:16（縦型）** でイベント告知動画を作る想定で進めます。
> 「9:16」を選んでプロジェクトを作成してください。

<a id="step3"></a>
## ✅ STEP 3　素材の読み込み

### 手順
- [ ] 左上の **「インポート」** をクリック
- [ ] 使いたい動画・画像ファイルを選択（複数選択OK）
- [ ] 左パネルに素材が表示されたら読み込み完了
- [ ] 素材をドラッグしてタイムラインに追加

### 💡 ポイント
```
複数ファイルを選ぶとき → Ctrlを押しながらクリック
並び順を決めたいとき  → タイムラインにドラッグする順番で順序が決まる
```

### 🎯 実際にやってみよう
> islaのイベント写真や動画クリップを3〜5個読み込んでみましょう。
> タイムラインに並べると、上から順番に再生されます。

## ✅ STEP 5　カット編集（不要な部分を切る）

NG部分・無言の間・撮影ミスなど、いらない部分を削除します。

### 基本のカット操作
- [ ] タイムライン上のクリップをクリックして **選択**
- [ ] 再生ヘッド（白い縦線）を切りたい場所に移動
- [ ] **Ctrl + B** で分割
- [ ] いらない部分をクリックして **Delete キー** で削除

### ⌨️ よく使うショートカット

| 操作 | キー |
|------|------|
| 分割 | Ctrl + B |
| 元に戻す | Ctrl + Z |
| 再生・停止 | スペースキー |
| 1フレーム移動 | ← / → |

### 🎯 実際にやってみよう
> インタビュー動画の場合：
> 1. 動画を再生して「あー」「えーと」などの余分な部分を探す
> 2. その前後でCtrl+Bで分割
> 3. 余分な部分をDeleteで削除
>
> ✨ コツ：完璧にやろうとせず、まず大きく不要な部分を削るところから始めよう

<a id="step6"></a>
## ✅ STEP 6　テロップ（文字）を入れる

### テキストの追加
- [ ] 左パネルの **「テキスト」** をクリック
- [ ] **「デフォルトテキスト」** をタイムラインにドラッグ
- [ ] プレビュー画面上のテキストを **ダブルクリック** して文字を入力
- [ ] 右パネルでフォント・色・サイズを変更

### 📌 テロップのタイミングを合わせる
- タイムライン上のテキストクリップを **左右にドラッグ** → 表示タイミングを調整
- クリップの **端をドラッグ** → 表示時間を長く/短くする

### 🎨 isla動画でのテロップ例

**インタビュー動画の場合**
```
例）話している内容をそのまま文字に起こす
　「islaに入ったきっかけは国際交流への興味でした」
　→ 話者の名前と所属も小さく入れると◎
```

**イベント告知動画の場合**
```
例）日時・場所・内容を順番に表示
　「📅 2024年〇月〇日（土）」
　「📍 〇〇大学 〇号館」
　「申し込みはプロフィールのリンクから！」
```

**ボランティア募集の場合**
```
例）活動内容・条件・連絡先を明確に
　「こんな人に来てほしい！」
　「語学力不問・週1回〜OK」
```

### 🎯 実際にやってみよう
> 動画の最初に「isla 〇〇イベント告知」というタイトルテロップを入れてみましょう。
> フォントは太めのものを選ぶと視認性が高まります。

<a id="step7"></a>
## ✅ STEP 7　BGM・効果音を入れる

### BGMの追加
- [ ] 左パネルの **「オーディオ」** をクリック
- [ ] **「ミュージック」** タブからBGMを選ぶ
- [ ] 追加ボタン or タイムラインにドラッグ

### 🔊 音量の目安

| 動画の種類 | BGM音量の目安 | 理由 |
|-----------|-------------|------|
| インタビュー動画 | 15〜25% | セリフを聞き取りやすくするため |
| イベント告知動画 | 40〜60% | 明るい雰囲気を出すため |
| vlog | 30〜50% | 映像に合わせてメリハリをつける |

### 音量の調整方法
- [ ] タイムライン上の音声クリップを選択
- [ ] 右パネルの **「音量」** スライダーで調整

### 🎯 実際にやってみよう
> islaのイメージに合った明るい曲を選んで追加してみましょう。
> 再生しながら「声が聞こえるか？BGMがうるさくないか？」を確認してください。

<a id="step8"></a>
## ✅ STEP 8　トランジション（場面切り替え）

クリップとクリップのつなぎ目に「切り替え演出」を入れます。

### 手順
- [ ] タイムライン上のクリップの **つなぎ目（｜マーク）** をクリック
- [ ] 左パネルの **「トランジション」** をクリック
- [ ] 使いたいトランジションを選んで **ダブルクリック**

### 📏 使い方のルール

| 動画の種類 | おすすめのトランジション |
|-----------|----------------------|
| インタビュー | なし or フェード（自然な切り替え） |
| イベント告知 | スライド・ズーム（動きがあって目を引く） |
| vlog | フェード・ディゾルブ（おしゃれな雰囲気） |

> ⚠️ **注意：** トランジションの使いすぎは逆効果。1本の動画で1〜2種類に統一しよう。

<a id="step9"></a>
## ✅ STEP 9　書き出し（完成動画の保存）

### 手順
- [ ] 右上の **「エクスポート」** をクリック
- [ ] 設定を確認する
- [ ] **「エクスポート」** をクリックして保存先を選ぶ
- [ ] 完成！

### ⚙️ 書き出し設定の目安

| 設定項目 | 推奨値 | 備考 |
|---------|--------|------|
| 解像度 | **1080p** | SNS投稿の標準 |
| フレームレート | **30fps** | 通常の動画はこれでOK |
| 形式 | **MP4** | どのSNSでも使える |

> 💡 データ容量を小さくしたいときは解像度を **720p** に下げてOK

---

## 🆘 困ったときは

| 症状 | 対処法 |
|------|--------|
| 動画に音声がない | タイムラインの音声クリップを右クリック →「ミュート解除」 |
| 文字が動画からはみ出る | プレビュー画面でテキストをドラッグして位置を調整 |
| 書き出した動画が重い | エクスポート時に解像度を720pに下げる |
| 操作を間違えた | Ctrl + Z で元に戻す |
| アプリが固まった | 保存してからCapCutを再起動 |

## 🗂️ 動画の種類別チェックリスト

### インタビュー動画
- [ ] カット編集で「えー」「あー」を削除した
- [ ] 話者名のテロップを入れた
- [ ] BGMを15〜25%に下げた
- [ ] 1080p・MP4で書き出した

### イベント告知動画
- [ ] 日時・場所・内容のテロップを入れた
- [ ] 申し込み方法（リンクなど）を最後に表示した
- [ ] BGMが明るく元気な雰囲気か確認した
- [ ] 1080p・MP4で書き出した

### vlog・活動記録
- [ ] 映像の順番がストーリーとして繋がっているか確認した
- [ ] BGMのテンポが映像と合っているか確認した
- [ ] テロップで場所や状況を補足した
- [ ] 1080p・MP4で書き出した
